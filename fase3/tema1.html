Etiquetas semánticas

CSS o Cascading Style Sheets, es un lenguaje de programación que permite aplicar estilos a los dinstintos elementos de las págins web, de modo que los títulos, listas y párrafos pueden verse igual en todas y cada una de las páginas.

Cada CSS se compone de tres partes: un selector, una propiedad y un valor, y su sintaxis es:
selector {propiedad:valor;}

  • Selector: Indica a qué la etiqueta html se aplicará el estilo
  • Propiedad: Indica a que propiedad de la etiqueta se aplicará el estilo.
  • Valor: valor de la propiedad
